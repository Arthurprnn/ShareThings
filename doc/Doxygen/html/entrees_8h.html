◆ chiffrer_mot_de_passe()

char * chiffrer_mot_de_passe ( char *  mdp)

Permet de chiffrer un mot de passe en utilisant la bibliothèque <crypt.h>.

Paramètres
char*Prend comme paramètre le char* retourné par: char* creer_mot_de_passe().
Renvoie
char* La fonction rend un char* qui est une chaine de caractère chiffrée.

< Nombre de séquences nécessaire.

< Permet d'orienter le chiffrement.

< On réserve l'éspace nécessaire en mémoire pour le mot de passe chiffré.

< On enlève le "\n" et on le remplace par "\0".

< On parcours le mot de passe en claire, on le chiffre par 8 caractères que l'on concatène dans cryptmdp.

< On parcours et récolte 8 caractères avant de les chiffrer, puis de recommencer.

< Pour chiffrer les derniers caractères présent.

< On return le tableau contenant le mot de passe chiffré.

◆ creer_mot_de_passe()

char * creer_mot_de_passe ( )

Permet de créer un mot de passe dynamiquement en remplaçant les caractères saisis par le caractère étoile.

Paramètres
voidNe prend rien en paramètre.
Renvoie
char* La fonction rend un char* qui est une chaine de caractères.

< On récupere un caractère saisie, sans que l'itilisateur ait appuyé sur entrée.

< On remplace le caractère saisie par une étoile.

< Et on afiche l'étoile.

< Tant que le mot de passe est inferieur à 32 et que l'on ne détecte pas que l'utilisateur appuie sur entrée.

< On réserve en mémoire une taille adapté à la longeur du mot de passe.

◆ getch()

int getch ( )

La fonction getch() est l'abréviation de getchar qui permet de lire un caractère ou une touche au clavier. Cette fonction est bloquante jusqu'à ce que l'utilisateur appuie sur une touche. Le caractère n'est pas affiché à l'écran.

Paramètres
voidNe prend rien en paramètre.
Renvoie
int La fonction retourne l'encodage du caractère récupéré.

< Permet de recuperer un caractère rentré au clavier sans avoir à appuyer sur entrée.
